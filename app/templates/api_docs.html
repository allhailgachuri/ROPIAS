POST /analyze

Main onset and irrigation classification endpoint. Uses aggressive caching and provides ML confidence scoring.


// Request
{
    "latitude": 0.28,
    "longitude": 34.75,
    "city": "Nakuru" // Optional: Overrides lat/lon if provided
}

// Response
{
    "location": { "address": "Nakuru, Kenya", "latitude": -0.28, "longitude": 36.07 },
    "onset": { "result": "False Onset", "confidence": 0.88 },
    "irrigation": { "status": "Irrigate Today", "moisture_percent": 22.4 }
}

POST /api/forecast

Provides a 7-day quantitative risk assessment array leveraging forward-looking NASA POWER forecasts.
